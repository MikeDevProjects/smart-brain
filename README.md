# SmartBrain

## Overview

This app lets you detect multiple faces at once 😱.

SmartBrain also takes advantage of authentication and session management using JSON web tokens

Created to play around with:

- Amazon Web Services,
- Face detection,
- JSON web tokens,
- authentication

... and lots more 😁!

## How to use

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
